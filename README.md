### 🚀 Featured Projects

#### 🎓 [S2dent](https://github.com/OusamaJamalEddin) &nbsp;·&nbsp; Blockchain-Based Academic Credential Wallet
My graduation project: a decentralized system for issuing, holding, and verifying tamper-proof academic credentials on Ethereum. I **owned the Flutter mobile app end to end** (for students and verifiers) within a 3-person team.
- Feature-first clean architecture (data / domain / presentation) with **Riverpod** and **Dio**
- On-chain features: **EIP-712** signing, **ECIES** decryption, and **WalletConnect** integration (Reown AppKit)
- Time-limited **QR generation** and a **live scanner** that verifies credentials against smart contracts in real time
- Shipped as **v1.0.1**
> **Stack:** Flutter · Dart · Riverpod · Dio · WalletConnect · Solidity · Node.js · PostgreSQL
